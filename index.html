What is LightCode?

Want to learn how to program? Check out this simple and cool modular lights project! We wanted to do something involving hardware and make something that would encourage people, especially children and teens, to learn how to code. But really, LightCode is a great project for everyone!

Here's an example of code that turns a single flower light on when it's dark, and turns it off when the sun's out!

Programming

LightCode uses a visual drag-and-drop programming language based off of MIT's Scratch language.
